# Social Media Performance Analysis

This collaborative team project analyzes social media performance data collected from four major platforms: Facebook, Instagram, LinkedIn, and X (Twitter).

The project combines multiple datasets into a unified dataset and performs Exploratory Data Analysis (EDA) to evaluate platform performance using key metrics such as Impressions, Engagement Rate, Audience Growth Rate, Response Rate, Post Reach, and Likes.

The analysis includes data cleaning, data exploration, statistical summaries, platform comparisons, and visualizations to identify trends, measure consistency, and generate actionable insights from social media data.

## Technologies Used

* Python
* Google Colab
* Pandas
* NumPy
* Matplotlib
* seaborn
* Excel

## Project Objectives

* Consolidate data from multiple social media platforms
* Compare performance metrics across platforms
* Analyze engagement trends over time
* Evaluate audience growth and responsiveness
* Generate visual insights to support decision making
